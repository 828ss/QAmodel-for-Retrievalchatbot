# QAmodel-for-Retrivalchatbot
## Chinese Retreival chatbot（中文检索式机器人）的QA模型  
### 建设计划： 
一、 基于tf-idf等特征的问答机器人  
二、 基于语义模型，如cnn，rnn等深度学习模型的问答机器人  
三、 基于ELMO的问答机器人  
四、 基于BERT的问答机器人
